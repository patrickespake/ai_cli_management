# Medical Laboratory Website

This project is a website for a medical laboratory specializing in blood tests.

## Tech Stack

*   **Frontend:** React

## Features

*   Information about the laboratory and its services
*   Online scheduling for blood tests
*   Patient portal to view test results
*   Contact information and location

## Development

To get started, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd <repository-directory>
npm install
npm start
```
