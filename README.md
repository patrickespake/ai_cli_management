# Medical Laboratory Website

This project is a website for a medical laboratory specializing in blood tests, built with React.

## Project Overview

This React-based web application will serve as the digital presence for a medical laboratory that focuses on blood testing services. The website will provide information about available tests, lab procedures, and facilitate patient interactions.

## Features (Planned)

- Laboratory services information
- Blood test catalog and descriptions
- Patient portal for test results
- Appointment scheduling
- Contact and location information
- Educational resources about blood tests
