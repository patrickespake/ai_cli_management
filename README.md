# Medical Laboratory Blood Test Website

This repository contains the source code for a modern, **React-based** web application that will power the public-facing website of a medical laboratory specialising in blood tests.

## Project Goals

1. Present the laboratory’s services, pricing and certifications in a clear, accessible way.
2. Provide patients and clinicians with educational resources about common blood tests.
3. Offer an intuitive appointment-booking flow and a secure patient login area (to be implemented).
4. Adhere to healthcare-industry best practices for accessibility, privacy and security.

## Tech Stack

• **React 18** with functional components and hooks
• **TypeScript** for type safety and self-documenting code (planned)
• **Vite** for a lightning-fast development server and build pipeline
• **React Router** for client-side routing
• **Tailwind CSS** for utility-first styling (or another modern CSS framework of choice)
• **Jest / React Testing Library** for unit & integration testing

> NOTE: This stack is a starting point and may evolve as the project matures.

## Getting Started

1. Clone the repository

   ```bash
   git clone https://github.com/<your-org>/medical-lab-website.git
   cd medical-lab-website
   ```

2. Install dependencies

   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server

   ```bash
   npm run dev
   ```

   The site will be available at `http://localhost:5173` by default.

## Scripts

| Command        | Description                            |
| -------------- | -------------------------------------- |
| `npm run dev`  | Start Vite dev server with hot reload. |
| `npm run build`| Create an optimised production build.  |
| `npm run preview` | Preview the production build locally.|
| `npm test`     | Run unit tests.                        |
